# Welcome to the Official StudyBox's App Repository

## Installation
First of all, make sure you have configured an Emulator/Device to your Computer.
### Requirements
> NodeJS >= 14\
> NPM >= 7

### Start Dev
```sh
$ npm install
$ npm run android
```
